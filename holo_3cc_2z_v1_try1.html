Two-Zero 3-Cycles — try1 (D=n−2, coupled)

    
Graph: Gr(n−2,n) — two "0"s and (n−2) "1"s.
E.g., Gr(4,6) — n=6 bits, two "0"s, k=4 "1"s. A = 001111.
A is fixed at the start vertex (00 followed by k 1s). Click any vertex to set B.

    
Layout: Node B with zeros at indices i, j (0-based) is placed at (i,j) on the 2D lattice. E.g. 00111 → (0,1), 10101 → (1,3). Edges drawn as arcs when they would overlap.

    
try1 — Step 5(1): Holo rectangle height $D=n-2$ (matches graph diameter for two zeros), not $\lfloor n/2\rfloor$.

    
Rectangle: $2\times(n-2)$ cells (bottom-indexed rows; labels $r$-1 / $r$-2 as in the $k=1$ app).

    
Shading (coupled — Step 5(2) option A): Let $z_1<z_2$ be 0-based zero positions. Set $i_{\mathrm{eff}}=z_1+z_2-1$ and apply the same odd/even $k=1$ rule as holog_comp_v8a.html to this single index. Then $S$ = number of shaded cells and $d(A,B)=(S+1)/2$ (verified by brute force for $4\le n\le 14$).

    
Verification (step 4): Union of two independent $k=1$ shadows does not satisfy $d=(S+1)/2$ in general. Using $D=\lfloor n/2\rfloor$ clips rows and breaks the coupled identity for $n\ge 5$.
